# GERAG: Graph-Enhanced Retrieval-Augmented Generation

GERAG is an advanced Graph RAG implementation that leverages a custom PyTorch architecture. It uses an **Improved Cross-Attention** mechanism to align text token embeddings with graph node spaces, incorporating **Contrastive Learning** and **Graph Regularization** to ensure retrieval is topographically consistent with the underlying knowledge graph.

## ⚠️ Important: Data Prerequisites
This repository **does not** include a graph construction pipeline. It is designed to work as a downstream reasoning and retrieval layer. It assumes you have already built a knowledge graph using **Microsoft GraphRAG**.

### Artifacts Setup
You must place the Parquet files generated by Microsoft GraphRAG into the project's data directory:
1. Generate your graph using the [Microsoft GraphRAG pipeline](https://microsoft.github.io/graphrag/).
2. Locate the output folder (usually `ragtest/output/`).
3. Copy the following files: `entities.parquet`, `relationships.parquet`, `communities.parquet`, and `text_units.parquet`.
4. Paste them into the `data/artifacts/` folder of this repository.
